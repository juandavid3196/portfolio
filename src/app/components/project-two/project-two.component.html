Events Manager for Pablo Tobon uribe Teather

The platform manages all activities for organizing theater events, including event creation and sub-event management. It integrates with the Calendar API for scheduling, manages user roles and permissions, stores legal documents, and presents performance metrics through graphical reports.

Authentication and authorization system

Ensures secure user login and access control, allowing only authorized users to access specific platform features based on their roles.

Events manager

A core module that allows the creation, editing, and organization of theater events, ensuring all activities are efficiently managed, from scheduling to execution.

Calendar to schedule new events

An interactive calendar that makes it easy to plan and schedule new events, helping to set dates and times visually while preventing conflicts with other scheduled activities.

Legal Documents manager

A tool for storing and managing legal documents required for event organization, such as contracts with performers and permits, ensuring compliance with all legal regulations.

Performance metrics

A system that tracks key metrics to measure the success of events, including attendance, ticket sales, and audience satisfaction, providing critical data for continuous improvement.

Reports section

A dedicated area for generating detailed reports on event performance, logistics, and management, offering valuable insights for strategic decision-making.

Responsive Web Design

Ensuring that both administrators and users can access and manage events from any device.
