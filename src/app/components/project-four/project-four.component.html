WEB APPLICATION
Social Media App

This platform replicates common social media features, allowing users to interact with friends through posts, comments, and reactions. Users can sign up and log in through various methods, create and share posts with images, comment on and react to others' posts, and receive friend requests. The platform also offers profile customization, options to block annoying accounts, and updates on friends' activities.

Authentication and authorization system by Clerk

Implements Clerk for secure user login and access control, supporting login with Google and GitHub accounts. It provides a user-friendly interface for managing accounts, editing personal information, and accessing the dashboard.

Cloudinary managenement

Utilizes Cloudinary to manage and upload images, enabling easy and efficient cloud storage and data handling for all user-uploaded content.

Hostinger and CloudPannel

Leverages Hostinger and CloudPanel for seamless database connectivity and secure storage of all platform information, ensuring reliable data management and easy access.

Prisma ORM

Uses Prisma as the ORM to handle all database requests, map database structures, and efficiently manage the system's data operations, ensuring seamless data interactions and streamlined queries.

React hooks

Incorporates the latest React hooks, including useOptimisticHook, useFormStatus, and useActionState, to enhance form handling, improve user experience, and provide real-time feedback on form interactions.

Stories

Enables users to upload temporary stories that remain visible for 24 hours, allowing them to share fleeting moments and updates with their audience.

Post Interaction

Allows users to engage with posts through likes, comments, and shares, fostering community interaction and engagement on the platform.

Personal profile page

A customizable page where users can manage their personal information, preferences, and interests. It allows users to update details, control privacy settings, and display key attributes.
