WEB APPLICATION
Characterization platform for disabled people

This system profiles individuals with disabilities at the University of Antioquia. It features customizable surveys with various question types and accessible interfaces for users with visual or hearing impairments. The goal is to collect data to improve decision-making on educational programs, infrastructure, and the academic experience.

Authentication and authorization system

Ensures secure user login and access control, allowing only authorized users to access specific platform features based on their roles.

Survey manager for creators

A tool that allows creators to design, organize, and manage custom surveys to collect data from users regarding their specific needs or characteristics.

Question bank

A centralized repository of pre-defined questions that can be used in multiple surveys, streamlining the creation process for surveys and evaluations.

Settings section

Provides users with options to customize their platform experience, including personal preferences and account settings.

Preview mode

Allows users to see a real-time preview of their surveys or forms before publishing them, ensuring everything appears as intended

Evaluation system

A system that automatically processes and assesses student responses to surveys, providing insights, feedback, or scores based on predefined criteria to help measure understanding or gather specific data.

Administration panel

A comprehensive dashboard for platform administrators to oversee users, manage content, and configure platform-wide settings and permissions.
